# ⚡ 5分钟快速启动指南

## 问题现象

```
业务错误 [5]: {code: "500", msg: "No static resource api/notifications/unread/count."}
```

---

## 🚀 一键解决 (只需3步)

### 第一步：启动后端服务

**打开PowerShell，输入:**
```powershell
cd "D:\健身房预约小程序\springboot"
mvn spring-boot:run
```

**等待出现:**
```
✅ Tomcat started on port(s): 8080
✅ Started SpringbootApplication
```

⏱️ 耗时: 约1-2分钟

---

### 第二步：重编小程序

**微信开发者工具中:**
1. 关闭当前预览窗口
2. 按 `Ctrl + R` 重新编译
3. 等待编译完成

⏱️ 耗时: 约1分钟

---

### 第三步：测试功能

**在微信开发者工具的模拟器中:**
1. 点击底部导航 "我的"
2. 点击 "系统通知"
3. ✅ 应该能看到通知列表

⏱️ 耗时: 约1分钟

---

## ✅ 成功标志

你应该看到:
```
✅ 通知列表页面加载正常
✅ 显示"全部/未读/已读"筛选标签
✅ 左上角显示未读数量角标
✅ 可以点击通知查看详情
```

---

## ❌ 如果仍有问题

### 快速诊断工具

**在"我的"页面向下滚动，点击 "🔧 API诊断"**

1. 点击 "1️⃣ 后端健康检查"
   - ✅ 正常 = 后端启动成功，继续下一步
   - ❌ 失败 = 后端未启动或端口错误

2. 点击 "3️⃣ 获取未读通知数量" (需先登录)
   - ✅ 正常 = 问题已解决！
   - ❌ 失败 = 参考详细指南
