# 健身房预约小程序

## 项目简介
这是一个健身房预约系统,包含 Spring Boot 后端、Vue3 管理后台和 UniApp 小程序前端。

## 技术栈
- **后端**: Spring Boot + MyBatis Plus + Redis + JWT
- **管理后台**: Vue3 + Element Plus
- **小程序**: UniApp

## 项目结构
```
├── springboot/     # Spring Boot 后端
├── vue3/          # Vue3 管理后台
├── uniapp/        # UniApp 小程序
└── database/      # 数据库脚本
```
